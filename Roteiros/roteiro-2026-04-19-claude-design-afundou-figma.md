## Bloco 1 — O que é o Claude Design e como funciona (2min–5min)

### O produto
- Claude Design é uma ferramenta experimental da Anthropic para criar visuais a partir de prompts de texto
- Gera: protótipos de interface, slides de apresentação, one-pagers, pitch decks, mockups
- Alimentado por Claude Opus 4.7 — modelo mais poderoso disponível ao público
- Disponível hoje para assinantes Pro, Max, Team e Enterprise

### O diferencial técnico
- Quando você termina um design, o Claude empacota tudo num **handoff bundle**
- Com um único comando, esse bundle vai direto para o Claude Code
- Resultado: o mesmo sistema que criou o protótipo também pode escrever o código para construí-lo
- Isso fecha o loop: **texto → design → código → produção**, sem sair do ecossistema

### O que isso significa na prática
- Antes: você descreve para um designer → designer prototipa no Figma → dev pega o Figma e constrói
- Agora (com Claude): você descreve → Claude Design prototipa → Claude Code constrói
- Não é necessariamente melhor em qualidade de design. Mas é **ordens de grandeza mais rápido** para validação de ideias

**Citar:** "Available in research preview" — ainda é experimental, mas já funciona bem o suficiente para o mercado reagir

## Bloco 2 — Por que a Figma perdeu bilhões (5min–8min)

### A reação do mercado
- Ações da Figma caíram no mesmo dia do anúncio do Claude Design
- Isso acontece porque o mercado entende algo que usuários comuns ainda não viram: a Figma depende de ser o elo entre designers e desenvolvedores
- Se a Anthropic cria uma ponte direta entre ideia e código, **o Figma perde o motivo de existir** para uma fatia enorme do mercado

### Quem vai ser afetado de verdade
- Startups em fase de validação: ninguém paga assinatura Figma para fazer 3 telas de MVP
- Criadores solo e solo founders: fluxo de trabalho completo por $20/mês no Claude Pro
- Pequenas agências: protótipos rápidos para apresentar para cliente

### Quem NÃO vai ser substituído (ainda)
- Designers profissionais que trabalham em sistemas de design complexos
- Times com Design System consolidado e handoff multidisciplinar
- Projetos com alta fidelidade visual e consistência de marca

### O padrão se repete
- Mesmo que aconteceu com editores de código quando o Copilot chegou: programadores medianos foram pressionados, programadores sêniors ficaram mais produtivos
- Com design vai ser igual: quem usa Figma para criar wireframes básicos vai ser pressionado; quem usa para orquestrar sistemas de design complexos fica (por enquanto)

## Bloco 3 — O que você deve fazer agora (8min–11min)

### Se você é criador de conteúdo
- Claude Design é uma mina de ouro para quem cria thumbnails, layouts de carrossel e mockups de conteúdo
- Você pode descrever em texto e receber um visual pronto para usar como referência ou base
- Combine com Canva (para polimento) ou passe direto para o Claude Code se você tem um site

### Se você é empreendedor ou solo founder
- Use Claude Design para validar a interface do seu produto antes de contratar alguém
- O handoff bundle para o Claude Code transforma protótipo em código real — isso comprime semanas de trabalho
- Oportunidade concreta: construir MVPs mais rápido do que nunca

### Se você é designer
- Aprenda a usar o Claude Design como ferramenta de ideação, não como ameaça
- Use para gerar 5 variações de conceito em 5 minutos e apresentar para o cliente
- Seu valor está na curadoria, no julgamento estético e na gestão do processo — isso ainda é humano

### A janela de oportunidade
- O produto está em research preview — quem aprender agora vai estar à frente quando virar GA (general availability)
- Criar conteúdo sobre Claude Design hoje ainda é território quase virgem em PT-BR
