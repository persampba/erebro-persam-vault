## IA e Claude — Novidades da Semana

### Claude Cowork — GA (Disponibilidade Geral)
O **Claude Cowork** está agora disponível para todos no Claude Desktop (macOS e Windows). Permite gerenciar tarefas via thread de agente persistente: o usuário define o objetivo, e o agente executa de forma assíncrona. Planos Enterprise têm analytics expandidos e controles de acesso por papel (role-based access control). O Cowork representa uma mudança conceitual: Claude deixa de ser "chat" e vira "gerente de projetos autônomo".

### Claude Code Desktop — Reconstrução Completa (15/04)
Anthropic reconstruiu o app desktop do Claude Code com:
- **Sidebar de sessões paralelas**: múltiplos projetos abertos simultaneamente
- **Layout drag-and-drop**: reorganização visual do workspace
- **Routines**: automações que rodam sem sessão ativa — bundle de prompt + repo + conectores

### Ecossistema Anthropic Completo em Abril 2026
| Produto | Função | Status |
|---------|--------|--------|
| Claude Design | Prototipagem visual a partir de prompts | Experimental (Pro/Max/Team/Enterprise) |
| Claude Code | Desenvolvimento com agentes | GA — Desktop reconstruído |
| Claude Code Routines | Automações sem sessão ativa | Lançado |
| Claude Cowork | Gerenciamento de tarefas com agente persistente | GA (macOS/Windows) |
| Claude Opus 4.7 | Modelo base mais poderoso | GA desde 16/04 |

### Principais achados
- **87% dos criadores** já usam IA em 2026 — adoção massiva

- **IA multimodal** consolidada: texto + imagem + áudio + dados estruturados em um mesmo modelo
